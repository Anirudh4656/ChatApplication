## ✅ Features

- 🔥 Real-time chat using Socket.IO
- 🔐 OTP-based email authentication
- 👥 1-to-1 chat
- 🤖 AI-chat ready interface(In Progress)
- 📱 Fully responsive UI
- ⚡ Optimized rendering & state management
- 🔄 Auto updates via WebSockets
- 🌍 Production deployment ready
